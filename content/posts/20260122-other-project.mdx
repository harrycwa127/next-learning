This Project is Fully self developed for Learning Vue.js with Option API. Implementing simple login, user management (without db, load data from json), custom toastWidget, routing.

- Login and authentication by role
- View Users (only admin)
- Search User (only admin)
- Add New User (only admin)
- Edit User (only admin)
- Delete User (only admin)
- Reset Users to Default
- Load Users from json
- Routing

## Satellite Delay Analysis

This project is Fully self developed for University FYP with python for calculate the delay of data transfer delay between ground station though LEO satellite.

### Features

- parameter setting with UI (setting data for orbit generation)
- set ground station and observation points etc data in file
- calculate delay for different path selection algorithm
- show result in 3D images
